Fusion 360: Student Start Guide

From Registration to your first 3D Print file

Category: CAD, Design, 3D Printing  |  Level: 100 - Introductory

Autodesk Fusion 360 is the industry standard for cloud-based CAD/CAM. The best part? It is free for students and educators. This guide will walk you through getting your license, creating a simple model, and exporting it for production.

Part 1: Account & Installation

Autodesk requires proof of enrollment to grant the educational license.

1. Create Account

  1. Go to the Autodesk Education Community.
  2. Select Fusion.
  3. Select your educational role (Student) and fill the Sign up for Autodesk educational access.
  4. If it ask you to login, create an account using your Student Email (example@student.fulbright.edu.vn).

2. Download & Install

  1. Once your education access is confirmed, click Access products to go back to the home page.
  2. Find Fusion 360 in the list.
  3. Click Download
  4. A small installer file will download. Run it. Fusion 360 downloads the bulk of the data during installation.
  5. When the installation is finished, sign in and start your creativity.

Part 2: Fusion tutorial - Example creating a cylinder

Once Fusion 360 is open, you will see a grid. Let's create a simple cylinder (50mm diameter, 20mm height).

Navigation: Hold Shift + Middle Mouse Button to orbit (rotate) your view. Scroll to zoom.

Method A: The Primitive Way (Fastest)

Fusion has built-in shapes for quick modeling.

  1. Go to the Create dropdown menu in the top toolbar.
  2. Select Cylinder.
  3. Select a plane to draw on (click the bottom/floor grid).
  4. Click the center point (Origin).
  5. Drag the mouse out and type 50 (for 50mm diameter). Press Enter.
  6. An arrow will appear. Drag it up or type 20 (for 20mm height).
  7. Click OK in the menu box.

Method B: Sketch & Extrude (The Professional Way)

This method is better because it allows you to modify the shape later easily.

  1. Click the Create Sketch icon (Green Plus sign) in the top left.
  2. Select the bottom plane (Floor).
  3. Press C on your keyboard (Shortcut for Circle).
  4. Click the center Origin point.
  5. Drag out, type 50, and press Enter.
  6. Click Finish Sketch (Top right checkmark).
  7. Press E on your keyboard (Shortcut for Extrude).
  8. Click inside the circle you just drew.
  9. Type 20 for the distance. Press Enter.

Part 3: Exporting Source Files

If you want to save your work to edit later or send to a teammate, you need the source files.

Exporting as .f3d (Fusion Archive)

This is the native format. It keeps the "Timeline" (history of your steps).

  1. Go to File (Page icon, top left) → Export.
  2. Name: My_Cylinder.
  3. Type: Select Archive Files (*.f3d).
  4. Click Export.

Exporting as .step (Universal)

Use this if sending the file to someone using SolidWorks, Inventor, or Blender.

  1. Go to File → Export.
  2. Type: Select STEP Files (*.step).
  3. Click Export.

Part 4: Exporting for 3D Printing (.stl)

3D printers cannot read .f3d files. They need a "Mesh" (a surface made of triangles). The standard format is STL.

The Modern Way (Save as Mesh)

  1. Look at the Browser tree on the left side of the screen.
  2. Find your object (usually named Body1 inside a Component folder).
  3. Right-Click on the Body or the Component name.
  4. Select Save As Mesh.
  5. Format: STL (Binary).
  6. Unit Type: Millimeter.
  7. Refinement: Set to "High" (this makes the circle look round, not like a hexagon).
  8. Click OK and save the file to your computer.
What's next? Take this .stl file and import it into PrusaSlicer (from our previous tutorial) to prepare it for printing.

Wrapping Up

You have successfully installed the software, modeled a part, and exported it for manufacturing. This "Sketch → Extrude → Export" workflow is the foundation of 90% of engineering design.
