Getting Started with Arduino

Learn to blink an LED in 5 minutes

Category: A, B, C  |  Level: 100 - Introductory

New to Arduino? You're in the right place. This guide contains all the information you need to get your first program working.

Introduction

Arduino is an open-source electronics platform based on easy-to-use hardware and software. It is intended for anyone making interactive projects.

Prerequisites

Before proceeding, make sure you have the following parts:

  • Arduino Uno Board (or compatible)
  • USB Cable (Type A to Type B)
  • Computer (Windows, Mac, or Linux)

Installing the Arduino IDE

To program the board, you need the official software (IDE).

  1. Download: Go to the official website: arduino.cc/en/software
  2. Select Version: Choose the version compatible with your operating system (Windows, Mac OS, or Linux).
  3. Install: Run the installer and follow the on-screen instructions.
  4. Launch: Open the application named "Arduino IDE".
Tip: We recommend downloading the latest version (IDE 2.x) as it includes features like auto-complete, which makes coding easier.

The Code: "Blink"

In the Arduino world, programs are called "Sketches". We will write a simple sketch to blink the built-in LED.

Every Arduino program has two main parts:

  • void setup(): Runs once to set things up.
  • void loop(): Runs forever in a loop.

Copy the following code into your IDE:

Arduino Blink Sketch
1void setup() { 2 // Initialize the LED pin as an output 3 pinMode(LED_BUILTIN, OUTPUT); 4} 5 6void loop() { 7 digitalWrite(LED_BUILTIN, HIGH); // Turn LED on 8 delay(1000); // Wait 1 second 9 digitalWrite(LED_BUILTIN, LOW); // Turn LED off 10 delay(1000); // Wait 1 second 11}

Uploading the Code

Once your code is written, it's time to upload.

  1. Connect your Arduino to your PC via USB.
  2. Go to Tools → Board → Arduino Uno.
  3. Go to Tools → Port and select your COM port.
  4. Click the Upload Button () in the top left corner.

If successful, you will see the text "Done uploading" at the bottom of the window, and your board will start blinking!

Wrapping Up

Congratulations! You have successfully programmed your first microcontroller.
